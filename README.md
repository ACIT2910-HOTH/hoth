### Build

A step by step series of building the environment:

Windows:
	1. git clone repository.link.git.
	2. Open command prompt to the repository folder.
	3. run npm install
	4. etc
	5. npm node server.js
		_Note: the server will be using port 10000_
	
Linux/MAC:
	1. git clone repository.link.git.
	2. Open command prompt to the repository folder.
	3. run npm install
	4. etc
	5. npm node server.js
	   _Note: the server will be using port 10000_
